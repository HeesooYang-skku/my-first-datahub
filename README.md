# Hello, SKKU! Streamlit App

This is a simple Streamlit web app created as a first practice project.

## Description

This app displays a title and a short message on a web page using Streamlit.

## Code

```python
import streamlit as st

st.title("Hello, SKKU!")
st.write("My first web app")
```

## Requirements

- Python 3
- Streamlit

## Installation

Install Streamlit with the following command:

```bash
pip install streamlit
```

## How to Run

Save the app code as `app.py`, then run:

```bash
streamlit run app.py
```

## Output

When the app runs, it displays:

```text
Hello, SKKU!

My first web app
```
